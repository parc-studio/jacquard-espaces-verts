Signature de mail

📧 Apple Mail (macOS)
  1. Clique Copier la signature
  2. Mail → Réglages → Signatures
  3. Sélectionne ton compte, clique + pour créer une nouvelle signature
  4. Décoche « Toujours utiliser la police par défaut »
  5. Clique dans la zone d'édition à droite et fais Cmd+V
  6. Renomme la signature, ferme les Réglages
📨 Gmail (web)
  1. Clique Copier la signature
  2. Gmail → ⚙️ Paramètres → Voir tous les paramètres
  3. Onglet Général, descend jusqu'à Signature
  4. Clique + Créer, donne un nom
  5. Clique dans la zone et fais Cmd+V (ou Ctrl+V)
  6. Sélectionne la signature comme défaut, sauvegarde en bas
📩 Outlook (web / desktop)
  1. Clique Copier la signature
  2. Web : ⚙️ → Voir tous les paramètres → Courrier → Composer et répondre
  3. Desktop : Outlook → Réglages → Signatures
  4. Crée une nouvelle signature, clique dans la zone et Cmd+V / Ctrl+V
  5. Sauvegarde, puis dans « Sélectionner les signatures par défaut » choisis-la
